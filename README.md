# RANDOM PASSWORD GENERATOR in C

## [English]

Using `/dev/urandom` for extra randomness.

**Q: Why Making this?**
**A:** Because I can't come up with a good password so I let the computer handle it.

**Q: You do know that computer can't make 'random', right?**
**A:** Yes, I'm fully aware of it. I'm just that lazy... don't worry, I will improve it later.

**Q: When do you planning to improve it?**
**A:** I don't know... I currently don't have skill in cryptography so... I have to read some book about it and learn.

**Features:**

- Cryptographically secure randomness (`/dev/urandom`)
- Configurable password length
- Bias-free character selection (rejection sampling)
